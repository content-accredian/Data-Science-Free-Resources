# Data Science Free Resources

Welcome to the **Data Science Free Resources** repository by [Accredian](https://www.accredian.com).[page:2]  
This repo curates high-quality, free learning materials to help students, professionals, and enthusiasts upskill in Data Analytics, Data Science, and Machine Learning.[page:2]

---

## About this repository

This repository is designed as a one-stop hub for data science learning assets, including:

- Role-based and skill-based roadmaps  
- Cheatsheets and quick-reference guides  
- PPT decks for teaching and internal training  
- Links and material for hands-on, project-based learning[page:2]

All resources here are free to access and aimed at helping you build end-to-end data skills, from fundamentals to advanced topics.

---

## Repository structure

The repo is organized into the following folders to make navigation easier:[page:2]

- `Roadmaps` – Career and learning roadmaps for Data Analyst, Data Scientist, and related roles.  
- `Cheatsheets` – Compact references for Python, SQL, statistics, machine learning, and more.  
- `PPT` – Presentation decks for workshops, classes, and internal enablement.  
- `index.html` – Landing page to browse or host these resources via GitHub Pages.[page:2]

You can click into any folder to explore and download the files you need.

---

## How to use these resources

- Start with the **Roadmaps** to plan your learning journey or teaching curriculum.  
- Keep **Cheatsheets** handy while coding, revising concepts, or preparing for interviews.  
- Use **PPT** decks to conduct sessions, bootcamps, or internal trainings.  
- Combine multiple resources (roadmaps + cheatsheets + slides) to build a complete learning path.

If you are new to GitHub, click on any file, then use the **Download** button to save it locally.
